## AFIT Admin

SPA para crear rutinas de gimnasio. (**En desarrollo**)

### Paquetes

- Angular 18
- Tailwind 3
- Angular PWA

### Funcionalidades

- Autenticación JWT
- Uso de Guard, interceptor, resolvers, services http
- Angular PWA
- Reactive Forms
- Custom Tailwind CSS
- Lazy loading (routing)

### Instalar y ejecutar app

```
npm install
ng server
```
