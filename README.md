## AFIT Admin

SPA para crear rutinas de gimnasio. (**En desarrollo**)

- El API esta en Django 5 [github.com/denuxs/gym_api](https://github.com/denuxs/gym_api)
- Usuario: admin / contraseña: endurance

### Paquetes

- Angular 18
- Tailwind 3
- Angular PWA
- Primeng

### Funcionalidades

- Autenticación JWT
- CRUD de usuarios
- CRUD de catálogos
- CRUD de ejercicios
- CRUD de rutinas
- CRUD de medidas
- Subida de Imágenes
- Notificaciones Push
- Traducción 118n

### Instalar y ejecutar app

```
npm install
ng server
```

### Development server

- Backend hosting en PythonanyWhere
- Frontend hosting en Netflify
